前端环境配置

一.nvm,node安装配置(最好先安装nvm,再安装node)

**1.**去官网https://github.com/coreybutler/nvm-windows/releases下载nvm-setup.zip压缩包并解压
注意：

1.安装路径不要包含中文！！！！！！！！！(血的教训)

2.安装过程中记住nvm路径

**2.**找到nvm里面的settings.txt，用记事本打开，并在里面添加如下内容，手动配置淘宝镜源：

node_mirror: https://npmmirror.com/mirrors/node/

npm_mirror: https://npmmirror.com/mirrors/npm/

npm config set registry https://registry.npmmirror.com/

目的：

-提高下载速度：淘宝镜像源的服务器位于国内，相比位于国外的官方源，国内用户访问时能大大减少网络传输距离，降低延迟。同时，淘宝镜像源拥有高带宽，能处理大量并发请求，在下载大型项目依赖或频繁更新依赖时，速度优势显著。

-增强稳定性：国内用户直接访问国外服务器易受网络波动、国际出口带宽限制等影响，导致连接不稳定。而淘宝镜像源在国内网络环境下运行更稳定，且针对国内网络进行了优化，减少了因网络波动造成的访问问题，数据丢包率也较低，能保证下载过程中数据的完整性和稳定性。

**3.**以管理员身份打开PowerShell，先测试是否安装成功nvm，输入 ：nvm -v，显示型号则表示安装成功，也是当前nvm版本
**4.**依次输入：

  • nvm list available: 用于查看可以安装的node版本 注意:尽量不要下载最新版本，以防出bug

  • nvm install 版本号: 安装指导版本node

  • nvm ls/nvm list: 查看已安装的node版本

  • nvm use 已安装版本号：用于切换当前使用的Node.js版本。例如，你在电脑上通过NVM安装了多个Node.js版本，如v14、v18、v20等，当你需要在不同项目间切换使用不同版本的Node.js时，就可以使用 nvm use 命令

  • node -v：检查是否安装成功node

​ 然后就大功告成啦，超级棒！
